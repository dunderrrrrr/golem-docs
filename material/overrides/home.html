What is Golem?


A flexible, open-source platform for democratised access to digital resources:
▶️ Lightweight protocol for building unstoppable applications, anonymous and censorship-resistant
▶️ Modular, cutting edge architecture
▶️ Fast and cheap micropayments with the GLM token on Ethereum’s Layer 2

With Golem Network, users can connect with ease and pay each other for sharing their unused resources. Golem’s democratized access combined with a unique peer-to-peer exchange creates an unstoppable ecosystem for a myriad of use-cases to be born, allowing software developers to leverage their creativity more than ever before.

Choose JS API if you:

✅ Want to use either Node.js or web browser to schedule the tasks

✅ Need more control over the deployment process on Golem Network (like adjust market strategy)
